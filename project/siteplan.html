Site Name

CarMatch

I chose CarMatch because the website helps users match with the best car for their needs. It can compare different new and used vehicles to help find the best match.

Site Purpose

The purpose of the site is to help people compare cars by viewing prices, features, and specifications. The site is intended to help users make informed decisions before buying a car.

Scenarios

  • A user wants to compare the price and features of a used Toyota Camry and a new Honda Civic.
  • A user is searching for a fuel efficient car and wants to compare options in their budget.
  • A first time car buyer wants to compare safety features, and overall cost between different models.

Color Schema

Primary Color: Dark Blue (#1f3a5f)

Used for headings, navigation, and section titles.

Background Color: Light Gray (#f4f4f4)

Used as the main background color to keep the layout easy to read.

Accent Color: Amber (#f5a623)

Used for buttons, links, and highlights to draw attention.

Text Color: Dark Gray (#333333)

Used for body text to create a modern look.

Typography

Heading Font: Arial, sans-serif

Arial will be used for headings because it is simple and easy to read.

Body Font: Helvetica, sans-serif

Helvetica will be used for paragraphs and comparison information.

Wireframe

Mobile View:

The mobile layout will use a single column design. The header will appear at the top, followed by car comparison content stacked vertically.

Desktop View:

The desktop layout will include a wider header and horizontal navigation. Car comparisons will be displayed side by side using to make it easier to compare vehicles.
